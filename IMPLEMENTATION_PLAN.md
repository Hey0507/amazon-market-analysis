# Amazon 市场大盘调研工具实施方案 (V2)

根据您的详细反馈，本项目旨在开发一个基于 Web 服务的 Amazon 市场数据 BI 看板。工具将通过对接 Sellersprite 的 MCP (Model Context Protocol) 节点提取原数据，进行特定业务规则的数据清洗（如仅保留月销量大于 200 的 ASIN），并将各个月份的维度数据汇总整理，最后通过交互式 Web 前端进行数据可视化呈现，并支持报表数据的原表 Excel 下载。

## ❗️ User Review Required: 方案可行性评估 (Feasibility Assessment)

综合您的意见，该方案在业务逻辑和技术选型上**完全合理且可行**。

> [!WARNING]
> **MCP 通信机制提醒**：
> 已确认 MCP 节点已在 Antigravity 中配置完成，可以直接通过工具名进行调用，无需额外编写连接逻辑。

## 🏗️ 架构设计与技术栈 (Web 服务端)

- **前端/全栈框架**：`Streamlit`。
- **数据处理引擎**：`pandas`。
- **MCP 客户端**：直接调用系统工具集。

## 🔄 数据流与业务处理逻辑

### 1. 数据获取 (Data Extraction)
- **输入**: 品类关键词 或 目标 ASIN 列表。
- **历史范围**: **过去 24 个月 (2年)** 的历史数据。
- **增量更新策略**: 仅抓取缺失月份。

### 2. 数据清洗与转化 (Data Cleaning & Transform)
- **异常/无用数据过滤**：**强制剔除月销量 <= 200 的 ASIN**。
- **价格分段计算**：`0~30`, `30~50`, `50~100`, `100~150`, `150+`。

### 3. Web Dashboard 展示 (BI 看板)
- **全局筛选器**：站点、品牌、月份、价格区间。
- **KPI Cards**：核心指标摘要。
- **可视化图表**：趋势、份额、价位段对比。
- **数据导出**：支持 Excel 下载。
